Pakiet dla innego modelu AI — KRS Guard Kalkulator Ryzyka
=========================================================

Zawartość:
1. KRS_Guard_reguly_i_zasady_funkcjonowania.xlsx — główne źródło reguł i tabel.
2. KRS_Guard_specyfikacja_funkcjonalna_logiczna_tekstowa.md — specyfikacja funkcjonalna/logiczna/tekstowa.
3. KRS_Guard_reguly_i_zasady_funkcjonowania.md — opis i pełny eksport arkuszy w Markdown.
4. folder csv/ — każda zakładka Excela jako osobny plik CSV, separator średnik.

Rekomendowane użycie:
- Excel traktować jako źródło nadrzędne.
- Markdown traktować jako wersję czytelną dla modelu AI.
- CSV używać do automatycznego przetwarzania tabel.
